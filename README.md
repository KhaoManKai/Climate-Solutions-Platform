## Climate Solutions Platform

(*This platform was completed during the third semester of a collegiate program.*)

a progressive web application displaying creative worldwide climate change ideas and projects. Users of the portal can search, add, sort, and learn about several projects aiming at mitigating climate change.

### Live Demo

Visit the live application: [Climate Solutions Platform](assignment-6-six-umber.vercel.app)

### Features

- Sort projects by sector (agriculture, electricity, transportation, etc.).
- See comprehensive material on every climate solution.
- User validation mechanism
- Managing project data on the administrative dashboard

### Technology Stack

- Node.js
- Express.js
- PostgreSQL
- EJS Templates
- Tailwind CSS
- Deployed on Vercel

### Development Journey

This project developed gradually, with each phase building upon the last:

- **a1**: Core file analysis capabilities
- **a2**: Initial web server implementation Express application with static content
- **a3**: Template integration and dynamic content
- **a4**: Database integration with PostgreSQL
- **a5**: Production deployment with user authentication
